Website is powered by open-meteo.com Website Weather API & API Docs Github Repository Air Quality API Open-Meteo combines local (1 km resolution) and global (11 km) weather models from national weather services. For every location on earth, the best forecast is available. National weather services include Deutscher Wetter Dienst (DWD), National Oceanic and Atmospheric Administration (NOAA), Meteofrance and Canadian Meteorological Center (CMC) In this project Open-meteo.com has provided the majority of the weather & air quality data for the project.
